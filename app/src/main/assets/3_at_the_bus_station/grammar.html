3 Grammar

Conditional with 'would'

Read the dialogue again. Look for the sentences with if + would.

“If you left home at 2pm you would miss the connecting bus.”

This is called 'Conditional 2'. In these sentences we talk about the results of things that are not likely to happen.

If you left home at 2pm
Event (that might not happen)

You would miss the bus
Result

In this case, the event is unlikely because we haven't decided when to leave home yet.

Even though we are thinking about the present or the future, we form Conditional 2 with the simple past and would + infinitive.

Practice

Make complete conditional sentences.

Example

0. get the 2 o'clock bus / arrive on time
If I got the 2 o'clock bus I would arrive on time

  1. take the number 1 bus / it be faster
  2. buy the tickets now / it be cheaper
  3. flight delayed/it be tight

Answers

  1. If I took the number 1 bus, it would be faster
  2. If I bought the tickets now, it would be cheaper
  3. If my flight was delayed, it would be tight

Conditional 1 vs conditional 2

Read the dialogue again and think about the following sentences.

A: If I miss it, can I get the next one?
B: If I missed it, could I get the next one?

What do you think the two sentences mean? What is the difference?

Answer

The speaker in the second sentence uses Conditional 2 (simple past + would + infinitve) because the event is more unlikely to happen.
