Grammar

Conditional 0 and Conditional 1

With Conditional 0, the result of the condition is always the same. It is formed with present simple... present simple.

If I come in the afternoon, I’m allowed to swim.

Conditional 1 is used to express a possible or likely future. It is formed with present simple... future.

If the pool isn’t open when I get there, I’ll go home.

Note that the condition (if the pool isn’t open) is always in the present simple, even if we are talking about the future. Only the main clause is in the future.

These are common mistakes in English.

Wrong: I’ll go to the pool on Sunday if it will be open.

Correct: I’ll go to the pool on Sunday if it’s open.


Wrong: I won’t do salsa lesson unless I will buy a season ticket.

Correct: I won’t do salsa lessons unless I buy a season ticket.

Conditional words

If is a common word to introduce a conditional sentence. We also use unless and as long as to introduce conditions. Can you find where they are used in the dialogue?

Which of these shows the meaning of unless and which means as long as?

  1. You can use the pool only if you can swim.
  2. You can’t use the pool, except if you can swim.

Answers

Notice how you use unless when one of the statements is negative, or implies a negative.

Your daughter can’t use the pool unless she wears armbands.

You can swim in the mornings unless there is swimming school (then you can’t swim).
